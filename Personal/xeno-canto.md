
# 🗂️ Xeno-canto

- **Nombre del Dataset**: Xeno-canto - Bird sounds from around the world.
- **Institución proveedora**: Xeno-canto Foundation for Nature Sounds.
- **Cantidad de registros**: el dataset contiene 54 registros.
- **Cobertura geográfica**: la cobertura geográfica de este dataset es global.
- **Cobertura temporal**: el dataset especifica una cobertura temporal desde 1900 hasta la actualidad.
- **Separador de campos utilizado en los archivos**: el separdor de campos utilizados es ",".
- **Codificación de caracteres (encoding)**: el encoding es UTF-8.
- **Tipo de licencia**: Creative Commons Attribution Non Commercial (CC-BY-NC) 4.0 License.
- **Frecuencia de actualización**: El dataset no proporciona una frecuencia de actualización.



## Para que sirve cada archivo



```bash
  📄 occurence.txt:
```
Contiene todos los datos del dataset.


```bash
  📄 multimedia.txt:
```
Contiene las grabaciones de los sonidos de los pajaros y la informacion del canto.
